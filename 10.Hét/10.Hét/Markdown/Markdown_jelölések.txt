Forráskód(részlet) beillesztése szövegbe `max(x-y)` és külön sorokban kiemelve:
```
known = {0:0, 1:1}

def fibonacci(n):
    assert(n >= 0), 'n must be >= 0'
    if n in known:
        return known[n]
    res = fibonacci(n-1) + fibonacci(n-2)
    known[n] = res
    return res
```
